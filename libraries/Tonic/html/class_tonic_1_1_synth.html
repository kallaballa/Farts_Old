◆ publishChanges()

ControlChangeNotifier Tonic::Synth::publishChanges ( ControlGenerator  input,
string  name = "" 
)
inline

Returns a ControlConditioner which accepts an input and a ControlChangeSubscriber (supplied by the UI). When the input value changes, ControlChangeSubscriber::messageRecieved is called. You would typically call this method inside a synth definition if you have a ControlGenerator whose value you want to make accessible to the UI thread. You then subscribe to these events by creating a subclass of ControlChangeSubscriber and passing a pointer to that object to Synth::addControlChangeSubscriber.
